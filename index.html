Automated 3D Model Creator using triangulation

I created this project using OpenCV that enables the creation of a 3D model from live video containing a known pattern for reference frame. The object to be modeled is placed within a predefined 3D volume w.r.t the reference frame. Subsequent images are taken with sufficient parallax and the 3D position of the points on the object is calculated using triangulation to enable the creation of a coarse model based on feature points extracted using FAST corners.
Repo: github.com/akkasuddin/Automated-Model-Creator
Report: goo.gl/2Az6x6

Age Of Quadrones

Led a team of 4 to generate an animated movie using OpenGL . I developed the game engine responsible for generating the required motion of the quadrotors and the camera motions that could be controlled using a script. I also wrote the code for generating the bombs and all the background code for handling collisions, different camera modes like follow-quad, look-at-quad, etc. Complex motions like moving a quad while also looking at another quad were also programmed into the engine.
Repo: github.com/akkasuddin/ageofquadrones
Video: www.youtube.com/watch?v=w8IaeyYAlLA

UAV Obstacle Avoidance using Stereo Vision and Fuzzy logic

Along with another team mate, I created a Fuzzy Logic based UAV Obstacle Avoidance program that uses stereo images to calculate depth from disparity . The depth information is then used as inputs to a fuzzy controller to generate commands that control the pitch and yaw of the UAV. The processing is GPU accelerated using CUDA on the NVIDIA Jetson TX1.
Repo: github.com/akkasuddin/UAV-Obstacle-Avoidance
Report: arxiv.org/abs/1705.04114

Domino Sorting Robot using CRS Catalyst-5 Robot

Led a team of three to program a CRS Catalyst-5 Robotic arm that sorts dominoes into two piles based on the number of dots on the dominoes. My contributions include the processing of images to segment out the dominoes and determine the positions, orientations and number of dots on the dominoes through the implementation of Circular Hough Transform , and connected component analysis . I also implemented the forward and inverse kinematics for the robot.
Report: goo.gl/TBdfyT
Video: www.youtube.com/watch?v=fNHNgufwQQ0

Daily Star Suicide Data Visualization

This is a project I worked on with The Daily Star Newspaper in Bangladesh to produce a visualization of suicides in the various districts between 2010 to 2014. The data was collected using a web scraper written in python and then classifying the articles collected based on the content. The visualizaiton was done by using d3.js and dc.js. This visualization is fully interactive and canbe used to study the trends of suicide in the country.
Link: Daily Star Visualization.
